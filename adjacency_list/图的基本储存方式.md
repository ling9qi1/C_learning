# 链式前向星（网络流）

```cpp
const int N = 1e6;
int idx, head[N], n, m, a, b;

struct Edeg {
    int from;
    int next;
    int to;
}e[N];

void add(int u, int v) {
    e[idx].from = u;
    e[idx].next = head[u];
    e[idx].to = v;
    idx++;
}

int main() {
    cin >> n >> m;
    for (int i = 0; i < n; ++i) {
        cin >> a >> b;
        add(a, b);
        add(b, a);
    }
}
